编辑工具

Typora

好用，目前收费，但可以白嫖（乐），具体下载地址请自行百度。还是推荐大家支持正版（叠甲）。

Visual Studio Code

目前在用，需要下载插件：

  • Markdown All in One
  • Markdown Preview Enhanced

Obsdian

用起来比较舒服。事常用工具

不止于Markdown，后续应该会开个坑说说这个工具（应该，咕咕咕）

Notion

原生支持Markdown，界面很简洁。且拥有很多强大的模板和功能
缺点：

  • 因为是上云的，且服务器在国外，所以打开比较慢（虽然我觉得速度还是可以接受）
  • 数据可能会丢失 (备份一下就好了) （切记一定要备份）
